# Product Overview

`anki-profiles-cc-sdk-codex` は、紙のプロフィール帳（プロフィール帳・交換ノート・クラス名簿など）に書かれた情報をデジタル化し、反復暗記に最適化するためのフロントエンドアプリケーションです。ユーザーは人物ごとの質問項目、回答テキスト、覚えておきたいトピックを整理し、スペースドリピティションのワークフローに乗せることで確実に記憶を定着できます。

## Core Capabilities
- プロフィール帳の項目テンプレート化と、人物ごとの回答レコード管理
- 回答テキストをヒント/回答形式のカードへ変換し、暗記タスクを自動生成
- 覚えたい優先度やカテゴリーによるフィルタリングとレビューキュー生成
- 暗記進捗・復習履歴の可視化と、次に復習すべきカードのサジェスト
- 将来的な SDK/外部ツール連携を見据えたデータエクスポート・インポート

## Target Use Cases
- クラスメイトやイベント参加者のプロフィールを素早く覚えたい学生・社会人
- アイドル/推し活ファンがメンバー情報やトリビアを整理、暗記したいケース
- コミュニティ運営者が会員プロフィールを共有し、メンバー同士の理解を深めたい場面
- SDK 化を通じて他サービスへプロフィール暗記体験を組み込みたい開発者

## Value Proposition
アナログで散逸しやすいプロフィール帳の情報を、学習体験に変換する点が差別化要因です。React + TypeScript を基盤に UI/UX を高速に迭代しつつ、Zustand などの軽量状態管理でシンプルに拡張できます。SDK 指向の構成により、暗記フローを既存サービスへ組み込むことも容易になります。
